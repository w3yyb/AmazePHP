# AmazePHP
AmazePHP  
A good choice for starting a PHP small project, especially for APIs.   


## features:  
configuration  
.env  
router    
mysql class  
uuid  
http client  
log   


## install:    
git clone https://github.com/w3yyb/AmazePHP.git  
composer install  

 ## run
 php -S 0.0.0.0:9080  public/index.php  
 open http://localhost:9080

 ## requirements:  
 php 8.1
